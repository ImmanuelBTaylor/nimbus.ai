## Getting Started: Join the ULE Revolution

### Prerequisites
- ULE Stack (download from [ULE.dev](https://ule.dev))
- Node.js (v18+)
- UNYTE CLI (`npm i -g unyte`)
- Osmium wallet (or compatible) for local $ULE testing

### Setup
```bash
# Clone (update repo name/path as needed)
git clone https://github.com/the-real-kodoninja/nimbus.ai.git
cd nimbus.ai

# Install ULE deps & setup
npm install
unyte init   # scaffolds ULE project structure

# Local blockchain + dev server (two terminals)
# Terminal 1
npx unyte node

# Terminal 2
npm run dev
```

Open <http://localhost:5173> --- connect wallet to localhost:8545.

Explore canisters, deploy marketplace logic with UNYSL, customize agents, and mint your first items on local ULE.
