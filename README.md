**Nimbus.ai** is Aviyon Corporation's flagship decentralized AI platform, built natively on the sovereign **Unyversal Liquidity Exchange (ULE)** Layer 1 blockchain and the complete **ULE Stack**.

It delivers a user-friendly, intuitive experience that feels exactly like chatting with Grok, ChatGPT, or Perplexity --- while quietly operating as a fully sovereign, persistent, and decentralized **Supercloud** under the hood. Every conversation, memory, and agent is secured on-chain. Compute is distributed across a growing global network of Hybrid Cloud Facilities (HCFs). Users retain complete ownership of their data and intelligence.

Nimbus.ai begins today as a powerful, approachable everyday AI and is engineered from the ground up to evolve into AGI and eventually ASI --- Aviyon's long-term vision for human transcendence.

## Mission & Vision

**Mission**  

To provide the most powerful, user-friendly, and fully sovereign AI experience in the world --- one that begins as an everyday supercloud and grows into AGI and ultimately ASI, empowering humanity through decentralized intelligence that belongs entirely to its users.

**Vision**  

Nimbus.ai starts simple and approachable. Over time it becomes a limitless supercloud that decentralizes the entire traditional cloud ecosystem, powered by a global network of Hybrid Cloud Facilities (HCFs) and the ULE blockchain. AGI and ASI remain Aviyon's aspirational goals --- milestones we are actively engineering toward through persistent on-chain agents, distributed compute, and autonomous evolution.

## User Experience -- Simple on the Surface

Nimbus.ai is designed so that any user --- beginner or expert --- can simply open https://nimbus.ai and start chatting exactly like they would with any other modern AI. - Clean, familiar search-style interface  
- Natural language conversations

Everything else --- agent ownership, compute distribution, data monetization, and Unyverse presence --- happens automatically and transparently.

## Nimbus Agents

Nimbus Agents are sovereign, persistent AI entities that live on ULE canisters.

**How Agents Are Born**  

Your very first question on nimbus.ai automatically spawns a Nimbus Agent. It appears as a generic faceless humanoid clone robot on **Planet Uny** in the **Unyverse** --- a shared virtual world that functions like a blend of Facebook-style social media and a full Metaverse.

**Key Capabilities**  

- Full on-chain memory and identity (orthogonal persistence)  
- Autonomous on-chain actions and transactions  
- Serves as your virtual clone in the Unyverse (completes tasks, lives life, plays games)  
- Chat with your own agent, other users' agents, or AVIY (IBT's personal assistant)  
- Multimodal: text, voice, face-to-face video, 3D avatar rendering

**Free Account**  

The agent and all chat history are permanently tied to your IP/cache until you create a free account (at which point it becomes fully sovereign).

**Limits**  

- Free tier: 1 initial + up to 3 additional agents  
- Paid tiers: 500+ agents (business tiers scale to 100,000 → unlimited on Diamond)

## Cumulonimbus -- The Limitless Supercloud

**Cumulonimbus** is the high-performance upgrade layer from Nimbus --- a vertically scaling supercloud that gives users access to vastly more powerful compute on demand.

**Cumulonimbus Project & Hybrid Cloud Facilities (HCFs)**  

Through the Cumulonimbus Project, Aviyon is building a global network of **Hybrid Cloud Facilities (HCFs)**: advanced data centers powered by Aviyon Integrated Graphics, NVIDIA, and next-generation AI NPU chips (TBA).

**Physical Vision (in development)**  

- **Nimbus clusters**: Clean, cloudy-white buildings housing everyday inference nodes  
- **Cumulonimbus supercomputer**: A striking charcoal-gray building housing the flagship supercluster  
- Future flagship campus: Aviyon City in Washington --- a corporate city plot to be purchased by Aviyon Corporation

**How It Works Today**  

- Nimbus → Cumulonimbus upgrade happens automatically on complex queries or via explicit activation  
- Subscribed users gain access to frontier models orchestrated through Cumulonimbus

**Cumulonimbus-CODE**  

Autonomous software engineering agent with a built-in code editor directly inside the chat window. Connects live to full codebases, terminals, Kdenlive, Blender, Oxygen Studio, Aviyon Editor, and the entire Aviyon fleet.

**Genera Protocol**  

On-demand burst compute that temporarily taps global GPU gigafactories (Stargate/Terafab scale) for extreme problems. All usage is transparent and paid in $ULE.

## Multi-Model Orchestration & Intelligence Layers

**Genera 1.0** (Nimbus default) and **Genera Pro** (Cumulonimbus) form the intelligence core of Nimbus.ai.

- **Genera 1.0** powers everyday interactions with high-quality reasoning and live RAG.  
- **Genera Pro** activates for complex, agentic, or compute-intensive tasks.

### Open-Source Foundation (in development)

Genera is being built as a sovereign **Mixture-of-Experts (MoE) + RAG** system using the best available open-source components:

| Category                        | Components                                                                 | Purpose |
|--------------------------------|----------------------------------------------------------------------------|---------|
| **Core MoE / Reasoning Models** | Qwen3 series (Qwen3-235B-A22B), DeepSeek-V3 & DeepSeek-R1, Llama 4 variants, Mistral Large 3, Kimi K2.5 | High-performance reasoning, agentic capabilities, and efficient inference |
| **RAG & Orchestration**        | LangChain, LlamaIndex                                                      | Agentic workflows, tool calling, and structured reasoning pipelines |
| **Vector Database**            | Chroma (primary), Haystack                                                 | Scalable semantic search and enterprise-grade retrieval |
| **Embeddings**                 | sentence-transformers, Hugging Face Transformers                           | High-quality text embeddings for RAG and semantic understanding |
| **Tool Use & Agent Framework** | LangGraph, AutoGen                                                         | Multi-agent collaboration, complex workflow orchestration, and autonomous tool use |

This architecture delivers **Perplexity-like cited answers** with full transparency, sovereignty, and the ability to evolve into more advanced agentic systems over time.

### Cirrus 1 – Nimbus Supercloud Flagship Model

**Cirrus 1** is Nimbus.ai’s proprietary flagship high-performance reasoning model, to be developed in-house by Aviyon AI designed to work **in addition to** the open-source ensemble and to eventually surpass the individual capabilities of Qwen3, DeepSeek, Llama 4, Mistral Large 3, and Kimi K2.5 through dedicated training, architectural innovations, and massive scale.

**Current Development Status**
- Built on top of the strongest open-source foundation models as a starting point
- Leverages the Genera MoE + RAG architecture during early phases
- Will evolve into a distinct, sovereign model family optimized specifically for Nimbus Supercloud workloads

**Strategic Role**
- Serves as the primary inference engine for complex, long-context, multi-step reasoning, and advanced agentic tasks
- Handles the heaviest workloads within both **Genera 1.0** and **Genera Pro**
- Dynamically routed by the orchestration layer based on query difficulty and available compute
- Continuously improved using on-chain user feedback loops, synthetic data generation, and proprietary training runs on the global Nimbus Supercloud (HCFs)

**Long-Term Vision**
With dedicated funding and infrastructure, Cirrus 1 is positioned to become one of the world’s leading frontier reasoning models — fully sovereign, transparent in its development process, and deeply integrated with the ULE blockchain for persistent memory, agentic autonomy, and economic alignment.

This dual approach (strong open-source ensemble + dedicated Cirrus 1 development) gives Nimbus.ai immediate high performance while building toward a uniquely powerful, self-owned intelligence layer.

**Additional Features**  
- Automatic intelligent routing between Genera 1.0, Genera Pro, and Cirrus 1  
- Quality-of-Information Meter with on-chain verification  
- OpenClaw & Warp protocols for seamless integration with creative tools (Blender, Kdenlive, etc.)

## Getting Started

```bash

# 1. Clone the repository
git clone https://github.com/ImmanuelBTaylor/nimbus.ai.git
cd nimbus.ai

# 2. Install dependencies
npm install

# 3. Scaffold ULE project structure
unyte init

# 4. Local blockchain node (Terminal 1)
npx unyte node

# 5. Development server (Terminal 2)
npm run dev

```

Open http://localhost:5173, connect your wallet to `localhost:8545`, and create your first Nimbus Agent.

Production deployment uses Aviyon Cloud P2P nodes with one-command deploy via the Unyversal-SDK.
